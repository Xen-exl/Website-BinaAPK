Keselamatan Tanpa Kompromi

Kami faham betapa pentingnya kod anda. Itulah sebabnya BinaAPK direka dengan prinsip keselamatan utama.

  • Proses berlaku dalam infrastruktur GitHub anda sendiri.
  • Kami tidak menyimpan kod sumber atau kunci peribadi anda.
  • Akses token hanya digunakan sementara untuk tujuan binaan.

Soalan Lazim

Adakah saya memerlukan pengalaman coding?

Tidak. Jika anda tahu cara memuat naik fail ke GitHub, anda boleh menggunakan BinaAPK.

Adakah token saya selamat?

Ya, token hanya digunakan untuk mengkonfigurasi GitHub Actions di repository anda. Kami mengesyorkan penggunaan fine-grained tokens.

Bolehkah saya publish ke Play Store?

Tentu sekali. APK yang dihasilkan adalah standard Android build yang boleh ditandatangani dan diupload ke Play Store.
